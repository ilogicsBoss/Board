Arduino IDE에서 사용할 수 있게 아이로직스에서 만든 통합 라이브러리이다.

본 라이브러리는 산업용 아두이노 내장 명령어를 만들기 이전에 사용하던 라이브러리이다. 참고만 해야한다.

1. CLCD: 캐릭터 LCD 라이브러리 (캐릭터 LCD에 PCF8574 I.C 내장된제품 전용)
2. DHT : 온습도 센서 연결 라이브러리
3. DS3231 : DS3231 RTC I.C 연결 라이브러리
4. IADC : 산업용 아두이노 제품의 아날로그 입력 라이브러리
5. IDAC : 산업용 아두이노 제품의 아날로그 출력 라이브러리
6. IPID : PID 라이브러리
7. IPWM : PWM 라이브러리
8. ISEG : 7세그먼트 연결 라이브러리 (7세그먼트는 4자리 이고 , 소수점 표현하는 제품과 시간표시 ":"를 표현하는 제품 2종이 있음. TM1637 I.C가 내장된제품 전용)
9. ITIMER : 타이머 인터럽트 라이브러리
10. M8A4RS : MPINO-8A4R-S 초기화 라이브러리
11. M8A8RS : MPINO-8A8R-S 초기화 라이브러리
12. M8A8R : MPINO-8A8R 초기화 라이브러리
13. M16A8R : MPINO-16A8R 초기화 라이브러리
14. M16A8R8T : MPINO-16A8R8T 초기화 라이브러리
15. MODBUS_RTU : Modbus RTU Slave 라이브러리
16. NTEMP : MPINO Series에서 NTC 온도센서를 연결했을때 라이브러리
17. WATCHDOG : 와치독 라이브러리
